What it does

Local system for home tasks, bills, and decisions.

  • Central place for maintenance, bills, and follow-ups
  • Clear timeline of past work and decisions
  • Reminders and next actions you control
  • Works on your machine without a cloud account
  • Open source and inspectable
  • Accessible from other devices on the same network

Install

Runs locally. No signup required.

  1. Get the code

    git clone https://github.com/Fuzzyslippers412/Mycasapro.git
cd Mycasapro

  2. Run the setup wizard

    ./mycasa setup

  3. Start everything

    MYCASA_API_PORT=6709 ./start_all.sh

  4. Open the app

    http://localhost:3000

  5. Use it on other devices

    Run once, then open on the same Wi-Fi:

    MYCASA_PUBLIC_HOST=<your-lan-ip> MYCASA_BIND_HOST=0.0.0.0 ./start_all.sh

    http://YOUR_LAN_IP:3000
