How it works

Tell it what you need, the agents organize it, and you approve the next steps.

Tell it

Share updates by chat, email, or connected tools.

Agents organize

Each agent files the work to the right place.

You review

See what is planned before anything happens.

It acts

Only what you approve gets scheduled or sent.

Fits into your existing tools

Use what you already rely on. Add more only if you want to.

Gmail WhatsApp Google Calendar Your model of choice

Nothing happens without you.

Control is the default, not a setting you hunt for.

Permission by permission

Each connection and action is explicitly approved and logged.

Reviewable decisions

You can see what was planned, why, and what actually happened.

Model flexibility

Use the built-in model or connect one that fits your privacy and cost needs.

Install

Runs locally. No signup required.

Requirements: Python 3.11+, Node.js 18+, Git

  1. Recommended (wizard)

    cd /path/to/mycasa-pro
cp .env.example .env
./mycasa setup

  2. Open the app

    http://127.0.0.1:3000

  3. Manual (two terminals)

    macOS / Linux 
    cd /path/to/mycasa-pro
cp .env.example .env
python3 -m venv .venv
source .venv/bin/activate
python install.py install

# Backend (terminal 1)
export MYCASA_API_BASE_URL=http://127.0.0.1:6709
export MYCASA_BACKEND_PORT=6709
export MYCASA_CORS_ORIGINS=http://localhost:3000,http://127.0.0.1:3000
python -m uvicorn api.main:app --host 127.0.0.1 --port 6709

# Frontend (terminal 2)
cd frontend
echo "NEXT_PUBLIC_API_URL=http://127.0.0.1:6709" > .env.local
npm install
npm run dev
    Windows PowerShell 
    cd C:\path\to\mycasa-pro
copy .env.example .env
py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1
python install.py install

# Backend (terminal 1)
$env:MYCASA_API_BASE_URL="http://127.0.0.1:6709"
$env:MYCASA_BACKEND_PORT="6709"
$env:MYCASA_CORS_ORIGINS="http://localhost:3000,http://127.0.0.1:3000"
python -m uvicorn api.main:app --host 127.0.0.1 --port 6709

# Frontend (terminal 2)
cd frontend
"NEXT_PUBLIC_API_URL=http://127.0.0.1:6709" | Out-File -Encoding utf8 .env.local
npm install
npm run dev

  4. URLs

    UI: http://127.0.0.1:3000

    API: http://127.0.0.1:6709

    API Docs: http://127.0.0.1:6709/docs

  5. Qwen OAuth (default)

    ./mycasa llm qwen-login

    Default model: qwen3-coder-plus

  6. Connectors (user-owned credentials)

    Gmail (gog)

    brew install doitintl/tap/gog
gog auth login

    WhatsApp (wacli)

    npm install -g @nicholasoxford/wacli
wacli auth

  7. Optional (LAN access)

    MYCASA_PUBLIC_HOST=<your-lan-ip> MYCASA_BIND_HOST=0.0.0.0 ./start_all.sh
